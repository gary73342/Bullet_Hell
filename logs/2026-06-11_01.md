# 2026-06-11_01 Reward 調整

## 變更內容

### 移除 X 軸對齊獎勵
- 刪除 `X_ALIGN_REWARD`、`X_ALIGN_THRESHOLD` 常數
- 刪除 `calculate()` 末段的對齊計算邏輯
- 原因：與排斥場信號衝突，且鼓勵暴露在敵機子彈路徑上

### 排斥場改為橢圓形衰減
- 原本：等向圓形 `d = hypot(dx, dy) / sigma`
- 現在：橢圓形 `d = sqrt((dx/sigma_x)² + (dy/sigma_y)²)`
- 子彈：`sigma_x=30, sigma_y=20`（原 25/25）
- 敵機：`sigma_x=30, sigma_y=20`（原 25/25）
- 目的：x 方向排斥範圍略大，鼓勵 agent 傾向往側邊閃避
